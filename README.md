# Resume-Builder-Java
Java Swing application for generating PDF resumes.

# Features
- user-friendly GUI
- Profile photo uploaded
- PDF generation using iText
- Input validation
- LinkedIn integration

# Technologies Used
- Java
- Java Swing
- iText PDF
